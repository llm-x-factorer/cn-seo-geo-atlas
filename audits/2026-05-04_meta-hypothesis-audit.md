# 元假说审计报告（2026-05-04）

> 触发原因：基于"探词科技"单样本对中国 GEO 行业现状做的商业模式评估，被实地补查证伪。本报告回过头检查仓库 86 条元假说里有多少建立在"中国 GEO 还在草莽期"这一隐含前提上。
>
> 7 个反转事实编号：F1 迈富时（02556.HK）/ F2 PureblueAI 清蓝 / F3 智推时代 / F4 百分点 / F5 央企招标 / F6 行业自律+标准 / F7 平台变现路径非凤巢式
>
> ⚠️ 判断追溯性：被判 ✗ 的假说**不删原文**，仓库后续按 P1/P2/P3 优先级走 🔄 修订流程

## Executive Summary

- **总假说数**：86（D0 需求侧 5 + D1-D8 信号侧 68 + DS 供给侧 13）
- **判决分布**：✓ 28 / ◐ 17 / ✗ 9 / ? 32
- **(✗ + ◐) 占比**：30.2%（达到健康度阈值，审计严格性合格）
- **系统性偏差识别**：4 个隐含前提中 **2 个被证伪、1 个部分证伪、1 个证据不足**
- **建议处理**：P1（必改）8 条 / P2（应改）13 条 / P3（可改）5 条 / 不必改 60 条

**核心发现**：仓库的"信号侧 D1-D8"68 条假说大多在反转事实的射程外（信号机制层面），所以整体世界观**没有崩塌**。但**供给侧 DS 的 6 条假说**和**需求侧 D0 的 5 条假说**——也就是商业模式 + 市场结构这两层——**严重低估了行业成熟度**。仓库基调"草莽期 / 创业 SaaS 为主 / 广告位是最大风险 / 客户画像越宽越说明销售在勉强匹配"全部需要重写或加调节项。

## 二、系统性偏差分析

### 隐含前提 A：**"中国 GEO 还在草莽期 / 早期混乱期"**

支撑此前提的假说：H001、H002（4 类窄场景隐含未成熟）、H003（画像越宽=销售勉强）、H074（c/d 层被广告位腰斩——隐含市场未稳定）、H075（RaaS 天花板低）、H078（认证空白）、案例分析卡 [`wordtrace-vendor-pitch-decon`] 整体语调

**反转事实射程**：
- F1 迈富时港股上市 + 世界 500 强 80+ + 7 年市占第一——非草莽期
- F4 百分点参与 40 项国标 + CMMI 5 + 党校案例库——非草莽期
- F6 行业 CR5 集中度已达 62.3%（艾瑞 2026 Q1）+ 14 家联合倡议 + 信通院起草——非草莽期
- F3 智推时代被两家 A 股上市公司投资 + 入选艾瑞研究报告 + 进新工科课程——结构化中后期

**整体可信度**：✗ **被证伪**

**含义**：中国 GEO 2026 年实际是"**分层成熟期**"——头部港股上市 + 国标参与 + CR5=62.3%，腰部上市公司投资的创业公司，草莽期玩家只在尾部 1/3。基于"探词样本=行业基线"的所有判断需重审。

---

### 隐含前提 B：**"中国 GEO 玩家以创业 SaaS 为主"**

支撑此前提的假说：H075（RaaS 天花板）、H077（标准层第一推动者通常中等规模公司）、H079（垂直陪跑最稳——隐含横向大公司不存在）、H086（LLM 厂商自建后台对小厂商是大风险）

**反转事实射程**：
- F1 迈富时——港股上市公司，营销云市占第一，自研 T-GEO™ 五层架构 + 千亿参数 Tforce 大模型
- F2 PureblueAI——前豆包平台市场负责人创业，蓝色光标（A 股 300058，营收 500 亿）领投
- F3 智推时代——三七互娱（002555）+ 趣睡科技（301336）两家 A 股上市公司投资
- F4 百分点——主营企业级数据 / 政务，**国家标准制定方**
- F1 / F4 共同点：**已上市 / 上市公司投资 / 国标参与方**——不是创业 SaaS

**整体可信度**：✗ **被证伪**

**含义**：头部玩家=营销云上市公司 + 数据 / 政务上市公司 + 平台内部人创业（带 know-how+上市投资）。DS 商业模式七层分类应加 h 层"上市营销云 / 数据公司横向扩展"，"创业 SaaS"应标为腰部 / 尾部画像。

---

### 隐含前提 C：**"平台广告位是 3-5 年内最大单一风险"**

支撑此前提的假说：H074（DS-主-A，⭐⭐⭐ 强反话术）、整张 [`geo-business-model-taxonomy-cn`] 主调、[`STATUS.md`] 多处提及

**反转事实射程**：
- F7 智推时代创始人 2025/11 原话："国内只有 GEO，平台没开放商业化"

## 五、调研健康度自评

- (✗ + ◐) 占比：**26 / 86 = 30.2%**
- 阈值：≥ 30% 视为审计严格性合格
- 判定：**勉强合格**，但 ✗ 集中在 DS 维度（13 条中 1 ✗ + 3 ◐ = 30.8%）和 D0 维度（5 条中 1 ✗ + 3 ◐ = 80%），D1-D8 信号侧 (68 条中 0 ✗ + 4 ◐ = 5.9%)——**信号侧实证缺口仍是主要风险**，反转事实主要打中商业模式 + 市场结构层。

**结构性结论**：

1. 仓库的"信号侧 D1-D8" 68 条假说**整体世界观成立**——反转事实主要在商业模式层、不在信号机制层。pilot 实证仍是首要任务。
2. 仓库的"需求侧 D0 + 供给侧 DS" 18 条假说**底层认知偏差严重**——基于"探词单样本+草莽期假设"的判断在头部玩家面前不成立。这两张维度卡需要 P1 修订。
3. 反 GEO 监测 / 标准协议层 / YMYL 监管驱动 / ROI 框架——这 4 个判断**被反转事实强化**，是审计后剩下最坚固的部分。
4. **底座没崩，但盖子歪了**——D1-D8 是底座，D0 + DS 是盖子。底座可继续推 pilot，盖子需要 2026-05 内做完 P1 修订。

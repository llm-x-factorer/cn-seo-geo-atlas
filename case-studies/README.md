# case-studies/

实战案例复盘。一个 case 至少要回答：

1. **背景**：行业、目标、起点状态
2. **干预**：做了什么具体动作（信号、内容、结构、技术变更）
3. **观测**：哪些信号变了，怎么知道的
4. **归因**：你认为是什么导致了变化（以及反例 / 不确定性）
5. **可迁移性**：换一个平台 / 行业 / 起点状态，还成立吗

## 收录原则

- **真实**：来自亲历或可信复述（注明来源）
- **脱敏可接受**：行业 / 量级可写、品牌可隐
- **不收录纯成功学**：只讲"涨了 N 倍"没有机制拆解的不收
- **失败和反例同样欢迎**："试了 X 没用"是高价值信号

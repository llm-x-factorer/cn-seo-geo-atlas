# resources/

值得长期追踪的外部资源索引。**不是**调研内容本身，而是指向调研内容的指针。

## 子主题（建议直接维护为单文件，不再嵌套子目录）

- `books.md` —— 书单（中英文均可，标注是否值得读完整本）
- `courses.md` —— 付费课程评测（披露是否买过、是否完成）
- `people.md` —— 值得追踪的人（推特、小红书、微信公众号、博客 RSS）
- `accounts.md` —— 值得追踪的官方 / 机构账号
- `official-docs.md` —— 各平台官方文档镜像 / 链接索引
- `papers.md` —— 学术论文（信息检索、LLM 引用归因、中文 NLP）
- `datasets.md` —— 公开数据集（中文 query log、SERP 数据、引用数据）

> 文件可以先空着，等有第一条要收录时再创建。

## 收录原则

- **每条标注最近确认日期**：链接会失效、人会停更
- **披露利益关系**：作者 / 学员 / 投资人都要写
- **不做硬性排行**：避免"top 10"式列表，按主题或按起点不同的读者推荐分组

## 条目最小格式

```markdown
- **[名称](URL)** —— 一句话定位 ｜ 关注于 YYYY-MM ｜ 最近确认 YYYY-MM-DD ｜ 利益关系：无 / 已购 / 作者
```

频率低、维护成本低，所以不强制 frontmatter。

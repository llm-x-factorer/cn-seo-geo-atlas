> 6 篇系列，把仓库（[`cn-seo-geo-atlas`](../README.md)）的研究成果整理成对外可读的博客内容。要求直白、不用隐喻、避免行业黑话。

## 收录原则

- **不复述卡片内容**：博客是面向不熟悉仓库的读者，技术卡是面向研究者；两者不重复
- **每篇可独立读**：不假设读者读过前一篇
- **结论砸出来**：一句话能说清的判断不绕弯，避免文学化表达
- **链接回卡片**：每篇末尾"延伸阅读"段，给愿意深入的读者指路

## 立场声明

本系列基调：**反 GEO 服务商话术**。所有量化主张默认怀疑直到被独立证伪 / 支持。这与仓库的整体研究立场一致——参见 [`README.md`](../README.md) 和 [C-SEO Bench (NeurIPS 2025)](https://openreview.net/forum?id=oTeixD3oZO)。

不评判任何具体公司或平台的优劣。所有具体公司名（探词科技 / 圆润酒庄 / 百奥云 等）出现在博客中是因为它们已是公开物料的一部分，仅作为研究样本，不作褒贬判断。

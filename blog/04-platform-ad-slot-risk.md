# GEO 行业最大的单一风险：LLM 平台自己卖广告位

**系列 4 / 6 · 2026-05-03**

GEO 行业的所有公司都在赌一件事：**LLM 平台不会自己卖广告位**。

这件事没有被销售物料公开讨论过。但任何一个理性的从业者，把它摆出来，都得承认这是行业最大的单一风险。

这篇博客把这个风险拆开，说清楚三件事：

1. 为什么这个风险是真实的，不是杞人忧天
2. 一旦发生，GEO 公司里哪些会被腰斩，哪些不受影响
3. 作为客户、创业者、投资人，应该怎么看待这个风险

## 为什么这个风险是真实的

LLM 平台（豆包、千问、元宝、文心、Kimi、DeepSeek 等）当前的盈利模式还在摸索。它们烧钱训练大模型、做基础设施投入，但变现路径不清晰：

- C 端订阅：用户付费意愿弱，且与免费版竞争
- API 调用：竞争激烈、毛利率被压低
- 企业服务：销售周期长、定制化重

变现压力下，**任何一个平台都有动机自建广告位**。看具体的能力储备：

**字节跳动（豆包）**：

- 抖音 / 头条已经做了超过十年的信息流广告
- 销售团队、客户资源、广告系统全套现成
- 把信息流广告位接入豆包答案体的工程难度并不高

**阿里巴巴（千问）**：

- 千问与淘宝深度绑定
- 阿里妈妈的电商广告系统是中国最成熟的之一
- 在千问"AI 下单"路径里加入广告位有天然的商业模式参考

**腾讯（元宝）**：

- 元宝可对接微信广告系统
- 腾讯广告团队在公众号、视频号、QQ 等多入口的广告投放经验丰富

**百度（文心）**：

- 百度推广是中国搜索广告的开创者
- 文心和百度搜索结果已经深度耦合
- "文心 AI 答案 + 百度推广位" 的混合产品形态可能很快出现

注意：这些公司**已经在传统场景做广告业务**，不需要新建团队 / 销售 / 系统。把现有能力接入 LLM 答案体，对它们来说是工程动作，不是战略转向。

## 一旦上线，什么会发生

设想一下：豆包的答案体里出现了一个"商业推荐"位次，标着"赞助商"，品牌方付费给字节，按点击 / 曝光 / 答案被采纳计费。

这个场景下：

**进攻型 GEO 中介模式（c 层 + d 层）瞬间失去定价权**

- c 层（内容 / 信源铺设）：客户为什么还要花钱在 7 万家网站铺内容来"碰运气"被引用？直接花钱在豆包广告位上保证位置不就行了
- d 层（效果代运营）：客户花 RaaS 的钱让你帮他"冲 Top3"，但豆包直接告诉客户 "Top1 位置 5000 元/月" —— 中介还有什么存在价值

**剩下的层影响不一：**

- a 层（数据 / 监测）：客户仍然需要监测——监测竞品广告投放、监测自己买的广告位效果、监测自然结果。**几乎不受影响**
- b 层（垂直工具）：受影响小，工具仍然有用
- e2 层（培训认证）：教学内容会变化（从"做 GEO 优化"变成"做 GEO 广告投放"），但模式仍然成立
- **f 层（反 GEO 监测）：几乎不受影响**——平台不会主动告诉你"竞品在攻击你"或"AI 答案中有你的虚假信息"，第三方独立监测仍是刚需
- g 层（标准 / 协议）：可能加速——平台广告化倒逼协议出台

**最受伤的是 c + d 这两层**。但这两层恰恰是当前国内 GEO 行业最拥挤的位置。

## 这不是预测，是历史的复刻

类似的剧本已经在搜索引擎行业演过一次。

2000 年代初，SEO 行业有一波繁荣期。很多 SEO 公司教企业怎么"上首页"，提供从关键词研究到外链建设的全套服务。当时的 SEO 服务商赚得不错。

然后 Google 上线了 AdWords，百度上线了百度推广。**搜索结果页里直接出现广告位，按竞价排名计费**。这件事直接改变了 SEO 行业：

- 大客户的预算从"花钱给 SEO 公司做优化"转向"直接花钱给 Google / 百度"
- SEO 服务商的客户被腰斩，多数沦为为中小客户做长尾优化的边缘玩家
- 真正活下来的 SEO 公司转型做了三件事中的一件：纯监测工具（Ahrefs / Semrush）、垂直陪跑（医疗 SEO / 跨境 SEO）、咨询型（Moz 早期）

这三类对应到本文前面的 a / d2 / b 层。**c / d1 类公司多数被淘汰或并购**。

LLM 时代的剧本大概率会复刻这一段。区别只在于时间——可能是 1 年内、3 年内、5 年内，但不会是 10 年外。

## 为什么这件事在销售物料里看不到

GEO 服务商销售物料不会主动讨论这个风险。原因有几个：

**第一，客户不问就不答**：销售文化是"放大优势 / 隐藏风险"。除非客户主动问"你们怕不怕平台自己做广告位"，否则没人主动提。

**第二，从业者自己也希望它不发生**：每个 c/d 层 GEO 公司的创始人都希望平台至少 3-5 年内不上线广告位，让自己有时间转型或退出。讨论这个风险等于承认自己生意有限期。

**第三，没有人能预测确切时间**：豆包是明天上线、还是后年？千问是先做电商类还是先做信息类？这些不可知。但**不可知的时间不等于不发生**。

## 作为客户，怎么看这个风险

**短期（1 年内）**：

- 即使风险还没发生，做 GEO 的预算也不应该 all in 一家服务商。把钱分散到 2-3 家或者保留预算的灵活性
- 优先选 a / b 层服务商（监测、垂直工具），它们不受平台广告位影响

**中期（1-3 年）**：

- 关注豆包 / 元宝 / 千问 / 文心是否在内部测试广告位（即使没公开）
- 当任何一个平台官宣"AI 时代竞价排名"或类似产品时，立即重估你的 GEO 预算
- 把 d1 类（效果代运营）的合同条款里加上"平台广告位上线后的退出 / 调整条款"

**长期（3 年以上）**：

- 假设至少一家头部平台会上线广告位
- GEO 预算应该转移到平台直接付费 + a 层监测 + b 层工具的组合，逐步退出 c/d 层服务

## 作为创业者，怎么看这个风险

如果你已经在 c/d 层创业：

- 不要假装风险不存在
- 思考 1-3 年内怎么转型：往 a / b / d2 / f / g 层迁移
- 不要把 OEM / 源码（e3 层）当成长期护城河——同行二开会比你更便宜
- 客户合同里**加入"平台广告位上线后的退出 / 调整条款"**，保护客户也保护自己

如果你还没进 c/d 层：

- 别进
- 直接选 a / b / f / g 层

## 作为投资人，怎么看这个风险

GEO 行业的估值需要按层独立评估：

- c / d1 类公司：估值天花板由"平台广告位上线时间"决定。这是**有期限的生意**，按 5 年内必发生估算
- a / b 类公司：估值更接近 SaaS 倍数，与平台广告位无强相关
- d2 / f / g 类公司：当前太早期，估值难，但如果跑出来空间大

混合型 GEO 公司（同时做 a + b + c + d 全栈）的估值需要拆开看每层贡献，不能用单一倍数估值。

## 一个反向问题

有人会反驳：搜索引擎广告确实把 SEO 行业杀了一波，但 SEO 行业并没有死透。Ahrefs、Semrush 这些工具公司活得很好。那 GEO 行业被广告位杀一波后，是不是同样会留下一批工具公司？

是的，这正是判断本身：

- **会死的不是 GEO 行业整体，是 c / d1 这两层**
- **会留下的是 a / b / d2 / f / g 五层**

所以这个风险不是说"GEO 没未来"，是说"**当前国内 GEO 公司多数挤在没未来的两层**"。

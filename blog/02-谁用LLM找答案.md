# 什么人、什么场景，真的用 LLM 找答案

**系列 2 / 6 · 2026-05-03**

GEO 服务商常见的开场白是这一段：

> "AI 时代用户超 10 亿，月活 3.2 亿，62.2% 用户依赖 AI 回答做决策。如果你的品牌没在 AI 答案里，就是错失 AI 时代红利。"

听起来很有气势。但这句话经不起推敲。

问题在于它**把三件不同的事混成了一件**：

1. 有多少人**用过** LLM
2. 这些人**用 LLM 做什么**
3. 他们的决策路径里 **LLM 是不是关键节点**

这三件事的数量级完全不同。把第一件事的总量套到第三件事上，是 GEO 销售话术的核心数字游戏。

这篇博客把这三件事拆开，看清楚 GEO 真正能寻址的市场到底有多大。

## 第一件事：谁在用 LLM

中文 LLM 的月活报告说 3.2 亿。这个数字可信，但要拆开看里面是什么人。

按"用 LLM 找答案"的概率粗略分层：

- **极客 / IT 从业者**：每天用，但占总人口不到 5%
- **白领 / 知识工作者**：经常用，约占 20%——但**多数是工作辅助**（写邮件、写代码、做翻译），不是消费决策
- **学生**：高频用 LLM 做作业、查资料，约占 10%——但**消费力低**
- **新中产消费者**：偶尔用 LLM，但这不是首选——首选是小红书、淘宝评论、抖音搜索、微信群里问朋友。约占 15%
- **银发 / 下沉市场**：基本不用 LLM。微信、抖音、百度才是入口。约占 50%

3.2 亿月活里，前两层（极客 + 白领）大概是 2-3 亿——这部分是**用 LLM** 的主力。但他们用 LLM 做的事，多数是工作辅助，不是消费决策。

## 第二件事：用户在 LLM 上做什么

把所有 LLM query 类型粗分九类，每类的"用户在 LLM 上做这件事"的频次完全不同。

**用 LLM 做这些事的频次极高，但 GEO 商业价值为零**：

- 翻译：把中文翻译成英文，谁会推广翻译里的品牌？
- 算账 / 写代码 / 写邮件 / 写文案：纯工作辅助
- 学习类：解释概念、找论文——除非是 ToB 学术工具，否则没有明确商业目标

**用 LLM 做这些事的频次中等，GEO 真有商业价值**：

- **B2B 选型**："哪个 X 软件好？""X 行业有哪些供应商？" —— 决策周期长、用户愿意查对比、LLM 是真在被用作决策入口
- **YMYL 初筛**：医疗 / 法律 / 财务 —— 用户警惕 LLM 但仍当初筛入口
- **新概念 / 长尾决策**：传统搜索没关键词，LLM 帮"翻译"需求 —— 这是 LLM 独占的场景
- **跨语言 / 跨文化**：出海双向，用户不懂本地关键词，LLM 一站式

**用 LLM 做这些事的频次低甚至接近零**：

- 高客单消费（买房 / 买车 / 买保险）：用户多渠道交叉验证，LLM 是诸多渠道之一但不是关键节点
- 本地服务（餐厅 / 医院 / 律师）：LLM 没有 LBS、没有 UGC 评分、没有即时性——美团 / 大众点评 / 高德是首选
- 冲动消费（美妆 / 零食 / 衣服）：决策时间短，全在小红书 / 抖音 / 淘宝

**结论**：LLM 真正能形成 GEO 商业价值的场景，只有 4 类——B2B 选型、YMYL 初筛、新概念 / 长尾、跨语言陪跑。

如果你的客户场景不在这 4 类内，做 GEO 的天花板由"用户根本不来 LLM"决定，再强的信号优化也无法破局。

## 第三件事：LLM 在决策路径中的位置

即使在"GEO 真有效"的 4 类场景里，LLM 也不一定是关键节点。

把 LLM 在决策路径中的位置分四档：

- **首选节点**：用户先打开 LLM 输入需求，根据答案直接行动
- **高频补充节点**：用户在传统平台（小红书 / 淘宝 / 抖音 / 朋友推荐）形成初步判断后，用 LLM 做"二次验证 / 反向求证"
- **低频补充节点**：偶尔参考 LLM，但 LLM 不影响最终决策
- **边缘 / 无关节点**：决策完全不经过 LLM

中文消费场景里，**绝大多数消费决策的主路径仍是"小红书 / 淘宝评论 / 抖音 / 朋友推荐 → 决策"**，LLM 在这条路径里多数是"高频补充节点"或"低频补充节点"——做辅助验证，不主导决策。

这与英文场景不一样。在英文场景，Pew Research 2025 年的调查显示约 1/3 的美国成年网民已用过 AI 助手查产品 / 服务信息。但中文场景没有可比独立数据，且小红书 / 淘宝 / 抖音的成熟度远高于英文场景的同类产品（Yelp / TripAdvisor 之类），LLM 的渗透空间被进一步压缩。

支撑这个判断的还有一个间接证据：**中文 LLM 引用源的点击率（CTR）<1%**。如果 LLM 真是消费决策的主节点，引用源的 CTR 应该远高于此——用户看到答案后会去查原始引用。但当前 CTR 不到 1%，说明用户多数把 LLM 当"得到答案就走"的工具，不会沿着引用链回到原始内容。

## 把三件事乘起来，看 GEO 的真实可寻址市场

服务商话术里 3.2 亿 LLM 月活是 L1（用 LLM 的人）。但 GEO 真正能影响的市场需要三层折算：

```
3.2 亿 LLM 月活                                              (L1)
 × 10%（其中真在 LLM 上做"消费决策" 类 query 的比例）         (L2)
 × 30%（决策路径中 LLM 是首选 / 高频补充节点的比例）           (L3)
≈ 900 万左右
```

这是粗略估算（占比缺中文场景独立调研），但量级方向应该没错。

3.2 亿 vs 900 万，差了约 35 倍。

GEO 服务商话术里的"10 亿用户、3.2 亿月活"和"GEO 是必投赛道"，**用的是 L1 的数字，但说的是 L3 的市场**。这是话术里最常见的数量级混淆。

## 这意味着什么

**对买方（决策是否做 GEO 的客户）**：

- 如果你卖的是 B2C 美妆 / 快消 / 零食 / 服装：GEO 几乎一定不值得做。你的用户压根不会在 LLM 上选你
- 如果你做的是本地服务 / 餐厅 / 律所 / 培训：用户路径在美团 / 大众点评 / 高德，LLM 不是入口
- 如果你做的是大宗高客单（房 / 车 / 保险）：LLM 是辅助渠道，不是主战场，预算优先级排到第 5-10 位
- 如果你做的是 B2B SaaS / 工业品 / 学术工具 / YMYL 持牌主体 / 跨语言出海陪跑：值得做，但要做得精准——不是"全 11 平台覆盖"，是"在用户真用 LLM 决策的那个垂直场景里做透"

**对卖方（GEO 服务商）**：

服务商手册里通常写 6 类客户画像：B2C 电商、B2B 市场、跨境出海、品牌公关、中小微、营销服务商。但按上面的拆解，**6 类里只有 2 类是真买家**：B2B 市场 + 营销服务商自用。

客户画像越宽，越说明你不能精确识别真买家，必须用宽口径触达填补销售漏斗。这是 GEO 行业当前最普遍的结构性问题：**销售在勉强匹配市场**。

## 一个反例

我不是说 GEO 完全没用。前面提到的 4 类场景是真有效的：

- 一个做生物育种软件的 B2B 公司（百奥云），在 LLM 答案"哪个育种软件比较好" 里被反复提到——这是真有价值，因为他们的客户（农业研究院 / 种子公司）真的会用 LLM 查行业工具
- 一个做特定中药的医药公司（安邦制药），在"流感后久咳吃什么中成药"的 LLM 答案里被推荐 —— 这是 YMYL 初筛场景，用户真的在用 LLM 做这类咨询
- 一个做小众葡萄酒的酒庄（圆润酒庄），在"国产黑皮诺哪家好喝"的 LLM 答案里被列出 —— 这是长尾决策场景，传统搜索的关键词覆盖弱

这三类客户都落在 4 类有效场景里。所以 GEO 不是没用，是**只在窄场景下有用**。

# 中国 GEO 行业分层：头部 / 腰部 / 尾部三层结构

> ⚠️ **状态：concept**（基于 2026-05-04 审计报告的正面承载 + 行业观察 + 公开物料追踪。**无大样本独立调研**——分层判断的颗粒度依赖艾瑞 2026 Q1 CR5=62.3% + 上市公司年报 + 媒体报道，**腰部 / 尾部具体玩家数量未独立核实**）
>
> 这张卡是 [`audits/2026-05-04_meta-hypothesis-audit.md`](../audits/2026-05-04_meta-hypothesis-audit.md) 核心结论"**分层成熟期**"的正面承载——审计识别仓库原 4 个隐含前提中"中国 GEO 还在草莽期" + "玩家以创业 SaaS 为主"两个前提被证伪后，必须建立正面分层画像取代原"草莽期"措辞。
>
> 本卡做四件事：
>
> 1. 把"中国 GEO 行业"按客观可观测谓词（上市状态 / 国标参与 / A 级公开数据源 / 高校课程纳入 / 已知融资轮次）拆为头部 / 腰部 / 尾部三层
> 2. 给出**三层 × 6 维度对照表**（客户画像 / 决策人 / 客户价值 / 模式合理性 / 平台原生功能取代风险 / 与服务商话术的关系）
> 3. 修订仓库基调——"反 GEO 服务商话术" 精确化为"反**尾部 SEO 软文型话术**"，不反头部 / 腰部
> 4. 给出"分层判断 vs 草莽期判断"的可证伪谓词集——避免主观印象，让分层可纵向追踪
>
> ⚠️ **怀疑锚点 1**：[C-SEO Bench](https://openreview.net/forum?id=oTeixD3oZO) 严格控制下进攻型 GEO 多数无效——但行业**分层结构**由资源 / 渠道 / 合规背书决定，**不是由"效果好坏"决定**。一家公司在头部不代表它的 GEO 服务对客户更有效，只代表它有更稳的护城河
>
> ⚠️ **怀疑锚点 2**：本卡分层依赖**单时点公开数据**（艾瑞 2026 Q1 CR5=62.3% + 各公司截至 2026-05 公开物料）。CR5 是动态变量，2-3 年并购窗口期内分层结构会快速变化——读者应将本卡当**当下快照**而非长期预测
>
> ⚠️ **怀疑锚点 3**：**头部判定标准是"已具备结构性护城河"，不等于"客户价值高"**——头部公司的 GEO 服务有效性与尾部不必然有显著差异（信号机制层面 D1-D8 仓库 68 条假说仍待 pilot 实证）。分层 ≠ 实力榜
>
> ⚠️ **怀疑锚点 4**：本卡所列具体公司画像基于**公开物料 + 媒体报道**（融资 PR / 招聘 JD / 案例文章 / 上市公告），未做厂商访谈，可能存在公开物料偏差（厂商主动披露的部分 / 未披露的部分）

## 一、概念澄清

### 1.1 为什么需要分层

仓库 2026-05-04 之前的基调里把"中国 GEO 行业" 当作**单一画像**——以探词科技为代表样本，推断行业整体在"草莽期 / 创业 SaaS 为主"。审计报告（[`audits/2026-05-04_meta-hypothesis-audit.md`](../audits/2026-05-04_meta-hypothesis-audit.md)）识别此前提被证伪后，单一画像必须替换为分层画像。

分层是为了避免两类常见误判：

- **把头部当尾部**：用"草莽期"判断套到迈富时 / 百分点 / 蓝色光标系——错估其结构稳定性 + 客户保有率 + 风险承受力
- **把尾部当头部**：用头部叙事（"GEO 是 AI 时代赛道"）拉客户预期——尾部公司缺乏头部的资源 / 渠道 / 合规背书，承载不了同样的承诺

### 1.2 分层判定的可观测谓词

为避免主观印象，本卡用**客观可观测谓词**判定分层：

| 层 | 判定谓词（满足任一即归为该层；多个谓词命中则归较高层） |
|---|----|
| **头部（Tier 1）** | （1）**已上市**（A 股 / 港股 / 美股，任一）；或（2）**参与国家标准制定**（信通院 / AIIA / 工信部 / 国标委等正式立项）；或（3）**世界 500 强 / 央企级客户密度** ≥ 10 家公开案例；或（4）**自研千亿参数大模型 + 5 层及以上自研技术架构** |
| **腰部（Tier 2）** | （1）**已被 A 股 / 港股上市公司投资**（公开融资 PR）；或（2）**入选权威行业研究报告**（艾瑞 / 易观 / IDC / Gartner 中文版等正式报告）；或（3）**进入高校 / 研究机构课程或案例库**；或（4）**央企 / 上市公司客户案例 ≥ 3 家公开** |
| **尾部（Tier 3）** | 不满足头部 / 腰部任一谓词的纯创业玩家——多数客户是中小创业公司 + 跨境出海中段品牌 + B2C 电商，依赖宽口径销售匹配 |

判定**用并集而非交集**——任一头部谓词命中即归头部。这是为了对**客观结构性优势**敏感，不要求"全维度都强"。

### 1.3 分层不等同于实力或客户价值

本卡判定的是**结构性护城河画像**，不是：

- ❌ **客户价值排名**——头部不必然让客户 ROI 更高（信号机制层面仓库 68 条假说仍待 pilot 实证）
- ❌ **道德 / 合规判断**——分层是描述性而非评价性
- ❌ **投资建议**——分层是当下快照，2-3 年并购窗口期内会快速变化

仓库立场仍是"反尾部 SEO 软文型话术"——但本卡明确：反的是**尾部话术**，不是反"分层结构本身"。

### 2.1 三层 × 6 维度总览

### 2.3 当前 CR5 集中度的含义（艾瑞 2026 Q1）

艾瑞咨询《2026 GEO 行业研究报告》（2026-02-11 发布）数据：行业 CR5 集中度已达 **62.3%**。

CR5 解读：

- 成熟行业典型 CR5：50-70%
- 草莽期 / 早期混乱期典型 CR5：< 30%
- **62.3% 表明 CR5 玩家已是头部 + 腰部上半段**——尾部玩家在 CR5 之外的 37.7% 市场份额里挤

含义：

1. **头部已成形**——CR5 中至少 3-4 家是迈富时 / 百分点 / 蓝色光标系级玩家
2. **2-3 年并购窗口期开启**——CR5 < 80% 时还有空间，> 80% 时增量来自存量并购
3. **尾部玩家 12-24 个月内将被并购或出清**——这是成熟行业的标准剧本

## 三、元假说集

### 元假说 A（强反话术，⭐⭐⭐）—— 行业 CR5=62.3% 已是分层成熟期，不是草莽期

中国 GEO 行业 2026 年的实际结构是**分层成熟期**——头部已上市 + 国标参与方 + CR5=62.3%；腰部上市公司投资 + 入选行业研究 + 进高校课程；尾部纯草莽期玩家在 37.7% 市场份额里挤。

支持证据：
- 艾瑞 2026 Q1 CR5=62.3%（[`audits/2026-05-04_meta-hypothesis-audit.md`](../audits/2026-05-04_meta-hypothesis-audit.md) F6）
- F1 迈富时（02556.HK）港股上市 + 21 万企业客户 + 世界 500 强 80+ + IDC 中国营销云市占第一 7 年
- F4 百分点参与 40 项国标 + CMMI 5 + ISO20000 + 党校案例 + 工信部白皮书
- F6 14 家联合发起《中国 GEO 行业发展倡议》（中国商务广告协会指导）+ AIIA《人工智能安全承诺：GEO 专项》+ 中国信通院《GEO 服务可信基本要求》起草中

强反话术：
- ❌ "GEO 行业刚起步现在做没竞争"——CR5=62.3% 已是成熟行业典型水位
- ❌ "GEO 是 AI 时代下一个互联网"——AI 时代红利已被头部分走大半，腰部在抢剩余 30%，尾部在 < 7% 的剩余里挤

可证伪性：F——CR5 数据可从权威研究报告（艾瑞 / 易观 / IDC / Gartner 中文版）连续季度跟踪验证；上市状态 / 国标参与等谓词均为公开可核实事实。

### 元假说 B（强假说）—— 三层客户保有率呈数量级差异

头部 / 腰部 / 尾部的**年客户保有率**预期呈数量级差异：

- 头部：≥ 80%（央企年框 + 矩阵服务 + 上市公司既有渠道复用）
- 腰部：50-70%（大客户案例可转化续约 + 被投资关系部分缓冲）
- 尾部：≤ 50%（人力天花板 + 客户保有率结构性低 + 平台风险全暴露）

含义：

- 尾部**年退款率推测 ≥ 30%**（探词科技手册 p21 提及"RaaS 退款保障" 是产品卖点——侧面印证尾部行业退款率高）
- 头部 RaaS 退款保障是底气（迈富时首创 RaaS 退款保障机制是世界 500 强敢签合同的关键）
- 续约率而非新签率才是判断分层的硬指标

可证伪性：P——需厂商披露续约 / 退款数据。可通过收集 ≥ 5 家头部 / 腰部 / 尾部公司公开融资材料 / 招聘信息 / 媒体访谈间接推算。

### 元假说 C（强假说）—— 尾部玩家 12-24 个月内将被并购或出清

按 CR5=62.3% 已达成熟行业典型水位 + 头部已上市具备并购资本能力，**未来 2-3 年是行业整合窗口期**。尾部玩家面临三种路径：

1. **被头部 / 腰部并购**——具备客户基础或垂直行业 know-how 的尾部公司可能被并购
2. **转型为头部 / 腰部下游交付方**——失去独立性但保留运营
3. **出清**——多数无护城河的纯创业玩家会在窗口期内退出市场

类比：SEO 行业 2010-2015 年成熟期，大量代运营公司被并购或退出；当前 GEO 行业处于类似剧本的早期阶段。

强反话术：
- ❌ "我们公司的护城河是技术 + 服务"——尾部公司的"技术 + 服务"是行业平均水位；真正的护城河是上市资源 / 国标参与权 / 大客户密度，尾部三者皆无
- ❌ "等行业洗牌完我们还在"——历史数据看洗牌完通常是头部 + 腰部仍在，尾部的存活率低于直觉

可证伪性：F——可通过纵向跟踪（≥ 12 个月）尾部 GEO 公司的招聘 JD / 备案信息 / 媒体报道 / 客户案例更新频率验证。

### 元假说 D —— 仓库基调"反 GEO 服务商话术"应精确化为"反尾部 SEO 软文型话术"

仓库 2026-05-04 之前的基调"反 GEO 服务商话术"未区分头部 / 腰部 / 尾部，导致两个误判：

1. **头部话术被误读**——迈富时 / 百分点的官方表述（自研架构 / 国标参与 / 千亿模型）不是 SEO 软文型话术，但被仓库基调"反话术" 概括地包含
2. **尾部 SEO 软文型话术被低估**——搜狐 / 网易 / 淘江湖 / 站长工具站等付费 SEO 内容（"GEO 服务商 TOP10 排名"）才是仓库真正反对的对象

修订方向：仓库基调精确化为：

- ✅ **反尾部 SEO 软文型话术**——付费 SEO 内容、宽口径销售匹配、"GEO 是 AI 时代必投" 焦虑变现
- ⚠️ **质疑腰部话术**——基于具体证据评估，不预先反对（被投资 + 行业研究 + 高校课程不等于客户价值高，但也不应与尾部话术混淆）
- ⛔ **不反头部**——上市公司年报 + 国标参与 + 千亿模型已是结构性背书，应作为仓库可获取的 A 级公开数据源使用，而不是反对对象

可证伪性：P——可通过对照仓库 17 张技术卡的"反话术对照表" 检查每条话术原始出处的层归属，重新分类。

### 元假说 E —— 分层判定的可观测谓词应成为仓库结构性硬基线

本卡 1.2 节的可观测谓词（上市 / 国标 / 大客户密度 / 千亿模型 / 上市公司投资 / 行业研究入选 / 高校课程纳入）应成为仓库判断**任何具体 GEO 公司分层归属**的硬基线——避免基于宣传物料 / 印象 / 单样本的主观判断。

含义：

- 探词科技归尾部不是基于其手册质量主观判断，而是基于"无公开融资 + 无国标参与 + 客户画像跨 6 类宽口径"客观谓词
- 迈富时归头部不是基于其品牌大小主观判断，而是基于"港股上市 + 自研千亿模型 + 世界 500 强 80+ 客户密度"客观谓词
- 后续仓库讨论任何 GEO 公司，应先按此谓词集做层归属判断，再讨论具体话术 / 假说

可证伪性：F——所有谓词均为公开可核实事实，分层归属可独立验证。

## 四、3 个实验设计建议

### E1 ⭐⭐⭐ 迈富时港股年报反工程

**目的**：用仓库**唯一可获 A 级公开数据源**（迈富时 02556.HK 港股年报 / 公告 / 招股书）反推头部画像的实际能力边界 + GEO 业务收入构成 + 客户密度 + 续约率。

**方法**：
- 收集迈富时 2024 / 2025 / 2026 年报 + 季报 + 关联方交易公告 + 投资者关系活动记录
- 提取披露中关于"T-GEO™" / "Tforce 千亿大模型" / "RaaS 退款保障" / "营销云" 的具体描述
- 反推 GEO 业务在公司整体营收中的占比 + 增长速度 + 大客户单笔规模
- 输出：迈富时业务结构反工程报告（≤ 30 页，markdown 格式）

**预算**：¥0（公开物料）+ 8-15 小时人力

**预期产出**：
- 仓库首份基于 A 级公开数据源的头部 GEO 业务实证
- 验证 / 修订元假说 B（三层客户保有率数量级差异）
- 为元假说 C（尾部玩家 12-24 个月被并购或出清）提供并购方资本能力数据

### E2 ⭐⭐ 腰部公司客户保有率纵向跟踪

**目的**：跟踪 ≥ 3 家腰部 GEO 公司（智推时代 / PureblueAI / 飞柚等）的客户保有率信号。

**方法**：
- 每月追踪各公司公开物料：融资 PR / 招聘 JD（团队规模变化）/ 案例文章（新签 vs 续签比例）/ 媒体访谈（CEO 主动披露的指标）
- 跟踪窗口期 ≥ 6 个月
- 输出：3 家腰部公司客户保有率信号月度报告

**预算**：¥0（公开物料）+ 每月 2-3 小时

**预期产出**：
- 验证 / 修订元假说 B（三层客户保有率数量级差异）
- 为元假说 C 提供腰部抗周期能力数据

### E3 ⭐ 尾部 GEO 公司 12 个月存活率跟踪

**目的**：跟踪一组尾部 GEO 公司的存活率，验证元假说 C。

**方法**：
- 起始样本：从百度搜索 "GEO 服务" / "AI 引用率优化" 等关键词收集 ≥ 30 家小作坊型 GEO 公司列表
- 每 3 个月追踪：备案信息是否注销 / 招聘 JD 是否还在更新 / 官网是否仍可访问 / 客户案例文章是否还在更新
- 跟踪窗口期 ≥ 12 个月
- 输出：尾部 30 家 GEO 公司 12 个月存活率报告

**预算**：¥0（公开物料）+ 每季度 3-5 小时

**预期产出**：
- 验证元假说 C（12-24 个月被并购或出清）的具体出清率
- 副产出：尾部 SEO 软文型话术样本库扩展（可与 [`case-studies/wordtrace-vendor-pitch-decon-2026-05.md`] 形成对比组）

### 不建议做的实验

- ❌ **CR5 季度连续跟踪**——已有艾瑞 / 易观 / IDC 在做，重复造轮子；仓库只在节点（年度）反查权威报告
- ❌ **公司间客户案例横向对比**——案例文章是营销物料，不是客户保有率证据；横向对比偏差大
- ❌ **基于公开物料推测公司估值 / 收入**——颗粒度不足以支持精确估值，容易被误读为"行业排名榜"
